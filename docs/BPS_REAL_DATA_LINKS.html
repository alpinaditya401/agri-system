Link Tabel BPS Utuh Agrilink

Daftar ini memakai halaman web resmi BPS berbentuk statistics-table, bukan endpoint WebAPI per potongan data. Satu kartu berarti satu konteks tabel BPS utuh yang bisa dibuka langsung di browser.

API key BPS hanya dipakai internal aplikasi untuk sinkronisasi otomatis. Untuk dibagikan di artikel, laporan, atau presentasi, gunakan link web BPS di bawah.
